O que são Redes de Computador?

Uma rede de computadores é um conjunto de dispositivos conectados entre si (computadores, roteadores, servidores etc...), de modo que possam trocar dados e compartilhar recursos (como internet, impressoras, arquivos), por meio de redes, é possível que computadores distantes comuniquem-se de forma rápida e eficiente.

O que é uma Rede Minima?

Uma rede mínima é a configuração mais simples possível de rede, basta dois dispositivos conectados entre si (por cabo ou via Wi-Fi) para já constituir uma rede, esse tipo de rede permite comunicação direta, troca de dados ou compartilhamento de recursos entre os dois dispositivos, serve como base para entender redes maiores.

O que é a Internet?

A Internet é uma rede global de redes, ou seja, a interconexão de milhões de redes de computadores espalhadas pelo mundo.

Ela é composta por:

  • Servidores e computadores espalhados pelo mundo.


  • Infraestrutura física e lógica: cabos (cobre, fibra óptica), roteadores, switches, data centers, provedores de acesso.


  • Protocolos de comunicação que padronizam como os dados são transmitidos.


  • Serviços e aplicações: páginas web, e-mail, redes sociais, streaming, etc...


O que é a WEB (World Wide Web)?

A Web (World Wide Web) é uma parte da Internet, o conjunto de páginas e sites acessíveis via navegador. Ou seja: Internet = infraestrutura + protocolos + serviços variados; a Web é um serviço de navegação de páginas usando protocolos específicos.

O que é o mapa de fibra óptica?

O “mapa de fibra óptica” é uma representação gráfica da infraestrutura física que conecta continentes e países por cabos de fibra óptica submarinos e terrestres, esses cabos formam a espinha dorsal da Internet global: grande parte do tráfego mundial — dados, vídeos, comunicação atravessa esses fios de luz.

O que é um protocolo de comunicação?

Um protocolo de comunicação define um conjunto de regras e padrões que controlam como dados serão transmitidos, recebidos e interpretados entre dispositivos de rede,Como Exemplo: como os pacotes são formatados, como são enviados, como erros são detectados, como respostas são feitas.

O que é o modelo Modelo OSI e o Modelo TCP/IP?

Modelo OSI serve como referência teórica para entender as funções de uma rede, é dividida a comunicação de rede em 7 camadas: física, enlace, rede, transporte, sessão, apresentação e aplicação.

Modelo TCP/IP é o modelo de comunicação usado na Internet, ele descreve como os dados são quebrados, transmitidos, roteados e reconstruídos entre dispositivos conectados, ele funciona como uma espécie de "manual" que todas as redes seguem para garantir que computadores diferentes consigam se comunicar perfeitamente, tendo a Camada de Acesso à Rede, Camada de Internet, Camada de Transporte e Camada de Aplicação.

O que são os protocolos IPv4 e IPv6?

IPv4: Versão tradicional do protocolo de Internet. Endereços de 32 bits, permitindo cerca de 4,3 bilhões de endereços únicos. Ainda é a versão mais usada.

IPv6: Versão mais recente. Usa endereços de 128 bits, permitindo muitos mais dispositivos e resolvendo o problema de escassez de endereços do IPv4.

Como é configurado o IP e o TCP em uma programação backend com node.js?

Você precisa escolher em qual endereço de rede o servidor vai funcionar e qual porta ele vai usar para receber as conexões. O IP identifica o computador na rede, podendo ser um endereço local, Já o TCP funciona por meio de portas, definidos por numeros, é por onde os dados entram e saem do servidor, como 3000, 5000 ou 8080. No Node.js, quando você cria um servidor você informa qual IP e qual porta ele deve “escutar”. Assim, o Node sabe onde ficar disponível e de onde aceitar conexões.

O que é o protocolo HTTP e o HTTPS?

HTTP (HyperText Transfer Protocol): Protocolo usado para a transferência de páginas web e dados na Web; define como o navegador solicita páginas do servidor e como o servidor envia respostas.

HTTPS: Versão segura do HTTP, que criptografa os dados trocados entre navegador e servidor — ideal para proteger informações sensíveis.
