O que é um SGBD? Tipos e Aplicações?

SGBD significa Sistema de Gerenciamento de Banco de Dados (em inglês, Database Management System – DBMS).
Ele é um software responsável por criar, organizar, armazenar, manipular e administrar bancos de dados, Em outras palavras, o SGBD é o intermediário entre o usuário (ou o programa) e os dados, garantindo que as informações sejam acessadas com segurança, rapidez e consistência

Exemplos de SGBDs conhecidos incluem: MySQL, PostgreSQL, Oracle, SQL Server, MariaDB e SQLite.
